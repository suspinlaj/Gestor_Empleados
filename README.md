# 👨‍🦱 **SusCompany: Gestor de Empleados**

**SusCompany** es una aplicación de escritorio orientada a la administración de recursos humanos. Este proyecto demuestra cómo una herramienta corporativa interna, tradicionalmente sobria y puramente funcional, puede transformarse en una experiencia visualmente atractiva y moderna mediante un buen diseño de UI.

## 📘 **Acerca de este proyecto**

Desarrollado como parte de la asignatura de **Desarrollo de Interfaces** en el segundo año de DAM (DAM 2). El desafío principal de esta práctica era construir una aplicación CRUD (Crear, Leer, Actualizar, Borrar) completa vinculada a la gestión de personal. En lugar de utilizar controles genéricos, diseñé una interfaz a medida utilizando formas orgánicas, una paleta de colores pastel muy cuidada y una distribución de la información clara y estructurada.

## 🔧 **Funcionalidades y Características**

- **Diseño UI Avanzado:** Interfaz limpia que utiliza formas personalizadas en XAML (como el panel lateral ondulado) e ilustraciones vectoriales para dar la bienvenida al usuario en la pantalla de inicio.
- **Gestión Completa (CRUD):** Panel de control lateral con acciones rápidas para la gestión de la base de datos: Buscar, Guardar (Añadir), Actualizar, Borrar y Limpiar formulario.
- **Visualización de Datos:** Estructuración de la información del empleado en bloques claros, mostrando su Departamento, Localización, Apellidos, Oficio, Salario, Comisiones y Fecha de Alta (utilizando *DatePickers*).
- **Sistema de Búsqueda Avanzado:** Sección dedicada para filtrar y buscar empleados dinámicamente según distintos criterios (por Apellido, por Oficio o por Salario), mejorando enormemente la experiencia de usuario (UX) al manejar grandes volúmenes de datos.

## 💻 **Tecnologías**

- **Lenguaje**: C#
- **Framework**: .NET MAUI (Multi-platform App UI) / XAML
- **Enfoque destacado**: Maquetación avanzada de vistas complejas, diseño de formas geométricas personalizadas, *Data Binding* y lógica de operaciones CRUD.

## ⚙️ **Estado del proyecto**

Finalizado. Proyecto educativo enfocado en el Desarrollo de Interfaces, con todo el flujo de gestión de empleados y la estética UI totalmente implementados.
